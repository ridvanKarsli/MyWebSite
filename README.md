# Ridvan Karsli Portfolio Website

Modern ve profesyonel bir portfolyo web sitesi. React, Material-UI ve Vite kullanılarak geliştirilmiştir.

## 🚀 Özellikler

- Modern ve responsive tasarım
- Material-UI bileşenleri
- Framer Motion animasyonları
- SEO dostu yapı
- Performans optimizasyonu
- Kolay özelleştirilebilir tema

## 🛠️ Teknolojiler

- React 19
- Material-UI
- Vite
- Framer Motion
- React Router
- TailwindCSS
- ESLint & Prettier
- Vitest & Testing Library

## 📦 Kurulum

1. Projeyi klonlayın:
```bash
git clone https://github.com/yourusername/ridvanskwebsite.git
cd ridvanskwebsite
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Geliştirme sunucusunu başlatın:
```bash
npm run dev
```

## 🧪 Test

```bash
npm test
```

## 📝 Kod Kalitesi

```bash
# Linting
npm run lint

# Formatting
npm run format
```

## 🏗️ Build

```bash
npm run build
```
